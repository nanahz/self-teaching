# 📚 《自学是门手艺》重造版
> 基于李笑来《The Craft of Self-Teaching》的个人重写与学习复盘项目


## ✨ 项目介绍
这是我**逐字重写、重构、深度复盘**《自学是门手艺》的学习项目。
不做简单的复制，而是：
- 用自己的语言重新理解、拆解书中核心观点
- 补全、优化书中的 Python 代码案例
- 加入自己的学习笔记、实践心得与拓展思考
- 全程用 Markdown 格式记录，形成一份可长期迭代、公开可见的学习作品

---

## 🎯 我的学习目标
1.  彻底吃透「自学」的底层逻辑，把方法内化成自己的能力
2.  系统巩固 Python 编程基础，练到能独立写出书中所有案例
3.  用「输出倒逼输入」，把这本书从「读过」变成「做过」
4.  打造一份可对外展示的个人学习作品，验证成长路径

## 🚀 如何使用
1.  直接在 GitHub 上浏览：每个 `.md` 文件都可以在线查看排版好的内容
2.  本地运行：克隆仓库到电脑，用 VS Code 打开即可编辑、预览
    ```bash
    git clone git@github.com:nanahz/self-teaching.git
    cd self-teaching
    ```
3.  跟着进度一起学：每完成一章，我都会同步更新提交，你可以看到完整的学习过程

## 💡 关于重造这件事
很多人学完《自学是门手艺》，只停留在「看懂了」，但我想更进一步：

这个项目就是我的「学习证明」——用最笨、最扎实的方式，把别人的知识，变成自己的能力。
